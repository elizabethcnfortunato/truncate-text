Truncate-Text
===============

A fast library using JQuery for truncating text.


Installing
-----

<script type="text/javascript" src="/truncate.js" ></script>

Usage
-----

<div id="myDiv">CONTENT HERE</div>
<script>
	$("#longText").truncate(1000);
</script>

Truncate-Text
===============

A fast library using JQuery for truncating text.


Installing
-----

&lt;script type="text/javascript" src="/truncate.js" &gt;&lt;/script&gt;

Usage
-----

&lt;div id="myDiv"&gt;CONTENT HERE&lt;/div&gt;
&lt;script&gt;
	$("#myDiv").truncate(1000);
&lt;/script&gt;

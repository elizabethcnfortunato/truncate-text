Truncate-Text
===============

A fast library using JQuery for truncating text.


Installing
-----

<code><script type="text/javascript" src="/truncate.js" ></script></code>

Usage
-----
<code>
<div id="myDiv">CONTENT HERE</div>
<script>
	$("#longText").truncate(1000);
</script>
</code>
